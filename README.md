Sistema de Cadastro de Estudantes

Descrição: API simples desenvolvida em Spring Boot.

==============================================================
Endpoints:

GET /api/sistema/info  
GET /api/cursos  
GET /api/cursos/{id}  
GET /api/estudantes/{matricula}  
GET /api/estudantes/curso/{curso}  

==============================================================
Como executar: Rodar a classe SistemaEstudantesApplication.

URLs:
01 - http://localhost:8080/api/sistema/info
02 - http://localhost:8080/api/cursos
03 - http://localhost:8080/api/cursos/{id}
04 - http://localhost:8080/api/estudante/{matricula}
05 - http://localhost:8080/api/cursos/{curso}
